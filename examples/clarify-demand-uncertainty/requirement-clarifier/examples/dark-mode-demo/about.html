关于本项目

项目简介

这是一个演示如何使用原生 HTML/CSS/JavaScript 实现夜间模式功能的教学项目。

项目展示了完整的前端主题切换实现，包括：

  • CSS Dark Class 主题切换方案
  • localStorage 状态持久化
  • 响应式设计
  • 平滑过渡动画

技术实现

HTML 结构

使用语义化 HTML5 标签构建页面结构。

CSS 样式

通过 body.dark-mode class 控制全局主题：

/* 浅色模式（默认） */
body {
    --bg-color: #ffffff;
    --text-color: #333333;
}

/* 深色模式 */
body.dark-mode {
    --bg-color: #1a1a1a;
    --text-color: #e0e0e0;
}

JavaScript 逻辑

使用 localStorage 保存用户偏好：

// 保存主题
localStorage.setItem('theme', 'dark');

// 读取主题
const theme = localStorage.getItem('theme');
