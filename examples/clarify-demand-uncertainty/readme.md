# 如果需求不明确则提问 (Ask Questions If Underspecified)

提出最少量的澄清问题，以避免错误的工作方向。

**作者：** FlyAIBox

##何时使用

当出现以下情况时使用此技能：
- 请求有多种合理的解释
- 成功标准、范围、限制或环境细节不清楚
- 在未澄清的情况下开始实施可能会导致错误的工作

## 功能

- 以易于浏览、易于回答的格式（多项选择 + 默认值）提出 1-5 个必须回答的问题
- 在提供所需的答案之前暂停行动（除非用户批准基于所述假设继续进行）
- 在开始工作之前重述确认的需求

## 安装

```
/plugin install trailofbits/skills/plugins/ask-questions-if-underspecified
```
