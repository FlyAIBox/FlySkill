**标题**：传统方式：三轮对话，650字提示词

**布局**：时间线展示

**内容**：

**第一轮对话（300字）**
```markdown
我上传了我的每周营销活动数据，希望分析其表现。

输入数据：
CSV 文件包含 微信广告、百度推广、抖音广告...

数据质量：
检查是否有缺失数据或任何异常情况。

漏斗分析：
为每个渠道计算以下两个指标：
• CTR（点击率）= 点击次数/展示次数
• CVR（转化率）= 转化次数/点击次数

并与以下历史基准进行比较：
渠道 | 目标 CTR | 目标 CVR
...（还有更多内容）
```

**标注问题点**：
- 🔴 需要每次重复输入
- 🔴 容易遗漏或写错公式
- 🔴 基准数据需要手动维护

### 第22页：C. 提炼判断逻辑

**标题**：第三步：把业务规则转化为if-else逻辑

**内容**：

**示例1：渠道差异化处理**
```python
IF 渠道 == "小程序推送":
    CTR = "N/A"（无展示次数）
    CVR = 正常计算
ELSE:
    CTR = 点击次数 / 展示次数

    状态 = "✓ 达标"

**示例3：预算调整规则（复杂决策矩阵）**

ELSE IF ROAS > 目标 AND CPA < 目标:
    分类 = "增加投入"
    调整幅度 = +20%
ELSE IF ...:
    ...
```

**教学要点**：
