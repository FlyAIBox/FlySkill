## 🎯 课程基本信息

### 课程定位
- **名称**：Agent Skill 重构传统SOP
- **时长**：80分钟（1小时20分钟）
- **难度**：入门实战
- **形式**：理论讲解（40%）+ 实战演练（60%）

### 目标学员
1. **产品经理、业务分析师**：希望用AI提升工作流程效率
2. **技术团队负责人**：需要建立团队级AI工作流标准
3. **AI应用开发者**：希望系统化管理AI提示词和工作流

### 前置要求
- ✅ 了解基本的AI对话交互方式
- ✅ 有一定的业务流程思维（了解SOP概念）
- ❌ 无需编程基础

## 📚 课程结构

### 七大部分

| 部分 | 标题 | 时长 | 核心内容 | PPT页码 |
|-----|------|------|---------|---------|
| 开场 | 从真实场景说起 | 5分钟 | 痛点场景 + 价值对比 | 2-4 |
| 第一部分 | 传统SOP的困境 | 15分钟 | 营销案例 + 三大痛点 | 5-11 |
| 第二部分 | Agent Skill的革命性突破 | 15分钟 | 核心理念 + 三层架构 | 12-18 |
| 第三部分 | 环境搭建与配置 | 15分钟 | Claude Code安装 + Skill安装 | 19-25 |
| 第四部分 | 深度解析Skill改造流程 | 20分钟 | 文件结构 + 6步改造法 | 26-36 |
| 第五部分 | 实战练习 | 20分钟 | 使用skill-creator创建Skill | 37-43 |
| 第六部分 | 最佳实践与进阶 | 8分钟 | 五大原则 + 团队管理 | 44-46 |
| 第七部分 | 回顾与行动计划 | 7分钟 | 核心回顾 + 三步行动计划 | 47-50 |

### 核心案例

#### 主案例：营销活动数据分析
- **传统方式**：3轮对话，1500字提示词，15分钟
- **Skill方式**：1轮对话，80字提示词，3分钟
- **效率提升**：5倍

#### 实战案例：需求澄清Skill
- **问题**：AI经常对模糊需求做错误假设
- **解决**：创建自动提问澄清的Skill
- **工具**：使用skill-creator创建

## 💡 课程特色

### 1. 以终为始的教学设计
- **痛点导向**：从真实烦恼出发
- **价值先行**：先展示效果，再讲原理
- **落地为本**：每个环节都有可落地的产出

### 2. 渐进式学习路径
```
观察（传统SOP问题）
   ↓
理解（Skill原理和架构）
   ↓
实践（改造 + 创建）
   ↓
应用（团队推广）
```

### 3. 双案例驱动
- **主案例**：营销活动分析（贯穿全课程）
- **实战案例**：需求澄清Skill（学员亲手创建）

### 4. 工具赋能
- **skill-creator**：降低创建门槛
- **标准化方法论**：6步改造法
- **可复制的模板**：SKILL.md结构模板
